# Talos 7 - Projekt SMiW

**Autor:** Szymon Wilczek
**Przedmiot:** Systemy Mikroprocesorowe i Wbudowane
**Rok akademicki:** 2025/2026

## Opis projektu
Talos 7 to programowalna, bezsterownikowa klawiatura makro (SudoPad) oparta na mikrokontrolerze Raspberry Pi Pico (RP2040). Urządzenie działa jako hybrydowy kontroler USB, łącząc funkcje klawiatury, myszy oraz interfejsu MIDI. Pozwala na automatyzację zadań i zwiększenie produktywności.

Główne cechy:
*   Brak konieczności instalowania sterowników (Driverless).
*   Konfiguracja przez przeglądarkę (Web Serial API).
*   Obsługa 7 klawiszy mechanicznych i wyświetlacza OLED.
*   Wsparcie dla systemów Windows, Linux, macOS.
